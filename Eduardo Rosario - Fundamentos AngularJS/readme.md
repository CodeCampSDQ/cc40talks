Fundamentos de AngularJS

En estos tiempos, los browsers han dejado de ser un simple programita que solo presenta de forma gráfica el código HTML a ser casi sistemas operativos; si no es que ya hay uno por ahí que lo es. Anteriormente la buena práctica era dejar toda la carga de una aplicación web en el servidor ya que el computador del usuario podría ser un poco lento, pero ahora, con tantos avances en la tecnología de información, este paradigma ha dado un giro de 180 grados.
AngularJS es un Framework creado por Google capaz de extender al máximo la capacidad del browser a través de JavaScript y HTML, haciendo de este la plataforma perfecta para el desarrollo de aplicaciones dinámicas y escalables.
Este permite al desarrollador ser más productivo con menos esfuerzo. Básicamente, esta librería extiende de forma significativa el HTML haciendo posible la arquitectura MVC del lado del cliente; dejando a un lado los PostBacks y permitiendo la creación de aplicaciones SPA.
Hablaremos de los fundamentos de AngularJS: como instalarlo, configurarlo y consumirlo utilizando Microsoft VisualStudio 2013, ASP.NET y C#. Repasaremos las tecnologías Json.NET, WebAPI y SignalR para su posterior integración con AngularJS.
