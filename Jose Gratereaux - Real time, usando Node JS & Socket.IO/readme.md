- updates in Real time, usando Node JS & Socket.IO
-------------------------------------------------------------------
En la práctica la Socket.IO abre una conexión al servidor que permite un intercambio de mensajes entre ellos (cliente-servidor) sin la necesidad de actualizar la página, es decir, sin la necesidad de solicitar una nueva petición al servidor. Esto hace que sea posible explorar el concepto de tiempo real en una aplicación, por lo que, básicamente, el cliente envía un mensaje y el servidor procesa y responde a través de una conexion directa.
